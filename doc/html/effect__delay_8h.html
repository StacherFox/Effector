Initialization function for the Delay Effect.

Parameters
*Spoints to an instance of the delay filter structure.
delayNumber of filter delay in ms.
alphaScaling factor applied to the delayed signal.
*pStateponts to the state buffer.
pStateSizeSize of the state buffer.
Returns
none.
Description
pStateSize is given by the equation: 
        pStateSize = (delayMax * SAMPLING_FREQUENCY)/1000 + 1.
or by: 
    delayMax = ((pStateSize-1) * 1000) / SAMPLING_FREQUENCY, in ms.
